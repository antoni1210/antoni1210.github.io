Stack

This was my first individual project, for which I had 6 days to build a game that could be rendered in a browser. The game had to be built using HTML, CSS and JavaScript/jQuery for DOM manipulation. The aim of the game is to stack animated blocks on top of one another. If the user fails to stack a new block precisely on top of the stack, the overhang is deducted from the blocks width, which the following block also inherits, thus making the game progressively harder.
